# Benutzerspezifischen Bericht ausführen

Sie können jeden Bericht ausführen, der noch nicht abgelaufen ist und derzeit nicht ausgeführt wird.

>[!NOTE]
>
>Sie können auch einen benutzerdefinierten Bericht ausführen, wenn Sie ihn [erstellen](report-create.md) oder [bearbeiten](report-edit.md).

1. Klicken Sie im Hauptmenü auf **[!UICONTROL Reports]** > **[!UICONTROL Custom Reports]**.

1. Klicken Sie neben dem Berichtsnamen auf **[!UICONTROL ...]** > **[!UICONTROL Run Now]**.

   Nach Abschluss des Berichts wird er an die in den Berichtseinstellungen angegebenen Ziele gesendet.

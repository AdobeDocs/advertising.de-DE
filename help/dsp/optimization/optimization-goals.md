# Optimierungsziele und Verwendung

| Optimierungsziel | Beschreibung | Verwendungszweck |
| -----------| ---------- | ---------- |
| [!UICONTROL Always Max Bid & Highest Clickthrough Rate] | Bei der Optimierung auf Paketebene wird bei der Budgetzuordnung Platzierungen mit den höchsten Clickthrough-Raten Priorität eingeräumt.<br><br>Bei Auktionsbewertungen wird der CTR Vorrang eingeräumt, wenn die Ausgabenziele erreicht werden. Das eingereichte Angebot ist immer die festgelegte [!UICONTROL Max Bid], aber der prognostizierte CTR-Schwellenwert wird strenger, wenn eine Platzierung gut ausgibt. | Kampagnentyp: Branding<br><br>Benchmark: Höchste Clickthrough-Rate<br><br>Anzeigentypen: Pre-roll, Display<br><br><b>Hinweis:</b> Verwenden Sie dieses Ziel, wenn Sie ein festes CPM-Ziel haben, das nicht überschritten werden muss, und ein CTR-Ziel, das maximiert werden muss. Setzen Sie das Max. Angebot auf das gewünschte CPM-Ziel, und DSP wird beim Versuch, das gesamte Budget auszugeben, die bestmögliche CTR erreichen. |
| [!UICONTROL Always Max Bid & Highest Completion Rate] | Bei der Optimierung auf Paketebene wird bei der Budgetzuweisung Platzierungen mit den höchsten Abschlussraten Priorität eingeräumt.<br><br>Bei Auktionsbewertungen wird die Abschlussrate priorisiert, wenn die Ausgabenziele erreicht werden. Das eingereichte Angebot ist immer das festgelegte Max-Angebot, aber der prognostizierte Schwellenwert für die Abschlussrate wird strenger, wenn eine Platzierung gut ausgibt. | Kampagnentyp: Branding<br><br>Benchmark: Höchste Abschlussrate<br><br>Anzeigentypen: Nur Pre-roll<br><br><b>Hinweis:</b> Verwenden Sie dieses Ziel, wenn Sie über ein festes CPM-Ziel verfügen, das nicht überschritten werden muss, und über ein Ziel mit der Abschlussrate verfügen, das maximiert werden muss. Setzen Sie das Max. Angebot auf das gewünschte CPM-Ziel. DSP wird beim Versuch, das gesamte Budget auszugeben, die bestmögliche Abschlussrate erzielen. |
| [!UICONTROL Always Max Bid & Highest Engagement Rate] | Bei der Optimierung auf Paketebene wird bei der Budgetzuweisung Platzierungen mit den höchsten Interaktionsraten Priorität eingeräumt.<br><br>Bei Auktionsbewertungen wird die Interaktionsrate priorisiert, wenn die Ausgabenziele erreicht werden. Das eingereichte Angebot ist immer das festgelegte Max. Angebot, aber der prognostizierte Schwellenwert für die Interaktionsrate wird strenger, wenn eine Platzierung gut ausgibt. | Kampagnentyp: Branding<br><br>Benchmark: Höchste Interaktionsrate<br><br>Anzeigentypen: Nur Mobile Interstitial<br><br><b>Hinweis:</b> Verwenden Sie dieses Ziel, wenn Sie über ein festes CPM-Ziel verfügen, das nicht überschritten werden muss, und über ein Interaktionsziel verfügen, das maximiert werden muss. Setzen Sie das Max. Angebot auf das gewünschte CPM-Ziel. DSP erzielt beim Versuch, das gesamte Budget auszugeben, die bestmögliche Interaktionsrate. |
| [!UICONTROL Always Max Bid & Highest Viewability Rate (Adobe – GroupM)] | Bei der Optimierung auf Paketebene wird bei der Budgetzuordnung Platzierungen mit den höchsten Sichtbarkeitsraten Priorität eingeräumt.<br><br>Bei Auktionsbewertungen wird die Sichtbarkeitsrate priorisiert, wenn die Ausgabenziele erreicht werden. Das eingereichte Angebot ist immer das festgelegte Max. Angebot, aber der prognostizierte Schwellenwert für die Sichtbarkeitsrate wird strenger, wenn eine Platzierung gut ausgibt. | Kampagnentyp: Branding<br><br>Benchmark: Höchste Sichtbarkeitsrate<br><br>Anzeigentypen: Interaktive Pre-roll-Funktion<br><br><b>Hinweis:</b> Dieses Ziel verwendet immer das Max. Angebot auf Platzierungsebene.<br><br>Wenn die Einstellung Sichtbarkeitsempfindlichkeit der Kampagne auf &quot;Standard (50 % der Anzeige für 2 aufeinander folgende Sekunden)&quot;festgelegt ist, wird der Standard für die Sichtbarkeitsmessung des Media Rating Council (MRC) für die Kampagne verwendet. Wenn die Kampagne auf &quot;Streng (100 % der Anzeige in Ansicht und Audio für 50 % Dauer)&quot;festgelegt ist, wird der GroupM-Standard für die Sichtbarkeitsmessung für die Kampagne verwendet. Idealerweise sollten Sie die Kampagneneinstellung mit dem Optimierungsziel und den Filtereinstellungen vor dem Angebot abgleichen. |
| [!UICONTROL Always Max Bid & Highest Viewability Rate (Adobe – MRC)] | Bei der Optimierung auf Paketebene wird bei der Budgetzuordnung Platzierungen mit den höchsten Sichtbarkeitsraten Priorität eingeräumt.<br><br>Bei Auktionsbewertungen wird die Sichtbarkeitsrate priorisiert, wenn die Ausgabenziele erreicht werden. Wenn die Kampagne auf &quot;Streng (100 % der Anzeige in Ansicht und Audio für 50 % Dauer)&quot;festgelegt ist, wird der GroupM-Standard für die Sichtbarkeitsmessung für die Kampagne verwendet. | [!UICONTROL Highest ROAS – Custom Goal] | (Nur auf Paketebene verfügbar) Bei der Budgetzuweisung werden Platzierungen mit der höchsten ROAS-Rendite (Return On Ad Spend, ROAS) priorisiert.<br><br>Bei Auktionsbewertungen wird ROAS Vorrang eingeräumt.

# Benutzerdefinierte Vorlageneinstellungen

| Registerkarte | Parameter | Beschreibung |
|--- |--- |--- |
| [!UICONTROL Date Range] | [!UICONTROL Date] | Der Datumsbereich, für den die Bedingungen für die Warnung ausgewertet werden sollen. Metriken werden über den gesamten Datumsbereich hinweg aggregiert ausgewertet. Die Optionen reichen von *[!UICONTROL Yesterday]* nach *[!UICONTROL Last 180 Days]*. |
| [!UICONTROL Filters] |  | Die Bedingungen für die Auslösung des Warnhinweises zu dem in der Variablen [!UICONTROL Scheduling and Delivery] Registerkarte. Die verfügbaren Spalten variieren je nach Entitätstyp (z. B. Kampagne oder Keyword). Alle Filter werden mithilfe der Booleschen Funktion UND verbunden, was bedeutet, dass alle angegebenen Bedingungen erfüllt sein müssen.<ul><li><p>Um einen Filter hinzuzufügen, klicken Sie auf ![Abwärtspfeil](/help/search-social-commerce/assets/arrow-down-expand.png "Abwärtspfeil") neben ![Hinzufügen](/help/search-social-commerce/assets/add.png "Hinzufügen") **[!UICONTROL ADD FILTER]** und führen Sie dann die folgenden Schritte aus:</p></li><ol><li><p>(Optional) Um die Spaltennamen nach Textzeichenfolge zu filtern, geben Sie die Suchzeichenfolge in das **[!UICONTROL ADD FILTER]** Eingabefeld.</p></li><li><p>Wählen Sie im Spaltenmenü einen Spaltennamen aus.</p></li><li><p>Definieren Sie den Filter für die Spalte:</p></li><ul><li><p>(Filter ohne Eingabefelder) Klicken Sie ![Abwärtspfeil](/help/search-social-commerce/assets/arrow-down-expand.png "Abwärtspfeil") neben dem zweiten Menü und aktivieren Sie dann die Kontrollkästchen neben den einzelnen einzuschließenden Werten.</p></li><li><p>(Metrikspalten, für die Sie Wertsteigerungen oder -rückgänge zwischen dem angegebenen Datumsbereich und dem vorherigen Zeitraum verfolgen möchten) Wählen Sie für den Operator *erhöht um* oder *verringert um* und geben Sie dann den Schwellenwert ein.</p><p>Wenn Sie beispielsweise die Variable[!UICONTROL Cost]&quot;. Wenn die Kosten einer Kampagne um 21 % steigen, sollten Sie eine entsprechende Benachrichtigung erhalten. Wählen Sie dann &quot;[!UICONTROL increases by]und geben Sie 21 in das Eingabefeld ein. Im [!UICONTROL Date Comparison Format] angeben, dass Sie an einer Änderung des Prozentsatzes interessiert sind (anstatt an einer Änderung der Währungseinheiten).</p><p>Wenn Sie diese Option auswählen, werden für jede reguläre Datenspalte zwei zusätzliche Spalten hinzugefügt. Anstatt beispielsweise nur eine Spalte für &quot;Klicks&quot;einzuschließen, enthält der Bericht Spalten für &quot;Klicks R1&quot;(für Bereich 1) &quot;Klicks R2&quot;(für Bereich 2) und je nach spezifiziertem Wert entweder &quot;Clicks Diff&quot;oder &quot;ClicksDiff (%)&quot; [!UICONTROL Date Comparison Format] (siehe unten).</p><p>**Hinweise:**</p><ul><li><p>Die Differenzspalte wird für abgeleitete Metriken nicht ausgefüllt.<p></li><li><p>Diese Funktion ist nicht für Classification-Spalten für Eigenschaftsnamen, ID oder Titel verfügbar.</p></li><li><p>Berichte, die große Datumsbereiche vergleichen, können länger dauern.</p></li></ul></ul><li><p>(Alle anderen Filter mit Eingabefeldern) Wählen Sie einen Operator aus dem zweiten Menü aus und geben Sie dann den entsprechenden Wert ein.</p><p>Wenn Sie beispielsweise die Spalte &quot;Klicks&quot;ausgewählt haben und nur Zeilen mit mehr als 100 Klicks zurückgeben möchten, wählen Sie &quot;größer als&quot;und geben Sie 100 in das Eingabefeld ein.</p><p>Je nach Datentyp können die verfügbaren Operatoren Folgendes umfassen: <i>größer als</i>, <i>kleiner als</i>, <i>gleich</i>, <i>contains</i>, <i>enthält nicht</i>, <i>beginnt mit</i>, <i>endet mit</i>, <i>kein Wert</i>, <i>hat Wert</i>, <i>before</i>, <i>after</i>oder <i>kein Datum</i>.</p><p>**Hinweis:** Bei Textwerten wird nicht zwischen Groß- und Kleinschreibung unterschieden. Wenn Sie z. B. nach Kampagnen filtern, deren Name &quot;loan&quot;enthält, können die Ergebnisse &quot;Consumer Loans&quot;und &quot;loan applications&quot;umfassen.</p></li></ol><li><p>Um einen Filter zu entfernen, klicken Sie auf ![Löschen](/help/search-social-commerce/assets/delete.png "Löschen") neben der Filterdefinition.</p></li></ul> |
|  | [!UICONTROL Comparing] | (Verfügbar, wenn der Warnhinweis Erhöhungen oder Verringerungen in einer oder mehreren Metrikspalten verfolgt (schreibgeschützt) Die beiden Datumsbereiche, für die Daten verglichen werden. |
|  | [!UICONTROL Date Comparison Format] | (Verfügbar, wenn der Warnhinweis Erhöhungen oder Verringerungen in einer oder mehreren Metrikspalten verfolgt) So geben Sie den Unterschied zwischen den Daten in den beiden Datumsbereichen aus:<ul><li><p><i>[!UICONTROL Variance]</i> (Standard) - Zeigt die Differenz als numerischen Wert an.</p></li><li><p><i>[!UICONTROL % Change]</i> — Zeigt die Differenz in Prozent an.</p></li></ul> |
| [!UICONTROL Scheduling and Delivery] | [!UICONTROL Name] | Der Warnungsname. Sie muss mindestens fünf Zeichen enthalten. |
|  | [!UICONTROL Trigger this Alert] [when] | Wie oft überprüft der Warnhinweis die angegebenen Bedingungsfilter und sendet E-Mail-Benachrichtigungen, wenn alle Bedingungen erfüllt sind:<ul><li><p>[!UICONTROL Daily at <*NN*> [AM|PM]]</p></li><li><p>[!UICONTROL Weekly on <*Day of Week*> at <*NN*> [AM|PM]]</p></li><li><p>[!UICONTROL Every month on <*Day NN*> at <*NN*> [AM|PM]]</p></li></ul>**Hinweis:** Dieser Wert wirkt sich nicht auf den Auswertungszeitraum aus. |
|  | [!UICONTROL Email Recipients] | (Nur vom Ersteller der Warnhinweisvorlage bearbeitbar; schreibgeschützt für alle anderen) E-Mail-Adressen, an die Benachrichtigungen gesendet werden sollen, wenn ein Warnhinweis erzeugt wird. Standardmäßig wird die Adresse des Vorlagenautors angegeben.<br><br>Um eine Adresse hinzuzufügen, geben Sie die Adresse ein und klicken Sie auf **[!UICONTROL Add]**. Um mehrere Adressen anzugeben, trennen Sie sie durch Kommas oder Leerzeichen oder fügen Sie sie getrennt hinzu.<br><br>Wenn der Warnhinweis bis zu 1000 Datensätze enthält, wird eine CSV-Version des Warnhinweises an die E-Mail-Nachricht angehängt. |

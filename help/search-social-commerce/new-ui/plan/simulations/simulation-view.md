# Anzeigen von Simulationsdetails

<!-- Needs subfeature tag "portfolios -->

*Beta-Funktion*

Sie können Simulationsdetails als Diagramm oder Tabelle auf dem Bildschirm anzeigen.

1. Klicken Sie im Hauptmenü auf **[!UICONTROL Plan]>[!UICONTROL Simulations]**.

1. Klicken Sie auf den Namen der Simulation.

   Die Simulation wird in der Diagrammansicht geöffnet.

1. (Optional) Führen Sie in der Diagrammansicht einen der folgenden Schritte aus:

   * Um die Metrik zu ändern, für die geschätzte Daten für jede Ausgabenzielebene (Schritt) angezeigt werden, wählen Sie eine neue Metrik aus dem **[!UICONTROL Y-Axis Metric]** Menü aus.

   * (Nur wöchentliche Simulationen) Geben Sie einen neuen **[!UICONTROL Target Mid Point]** ein, um die prognostizierten Daten mit diesem Wert anzuzeigen.

   * (Nur wöchentliche Simulationen) Wählen Sie im Menü **[!UICONTROL Constraints]** aus, ob die *[!UICONTROL With Constraints]*, *[!UICONTROL Without Constraints]* oder *[!UICONTROL With and Without]* angezeigt werden sollen.

1. (Optional) Um die Tabellenansicht zu öffnen, klicken Sie auf die Registerkarte **[!UICONTROL Table View]** .

1. (Optional) Um die schreibgeschützten Einstellungen für das entsprechende Portfolio anzuzeigen, klicken Sie oben rechts auf **[!UICONTROL Portfolio Settings]** .

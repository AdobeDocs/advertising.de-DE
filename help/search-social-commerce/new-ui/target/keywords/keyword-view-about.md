# (Neue Benutzeroberfläche) Über die [!UICONTROL Keywords]

*Beta-Funktion*

Die Ansicht [!UICONTROL Manage] > [!UICONTROL Keywords] umfasst zwei Registerkarten:

* Auf der Registerkarte [!UICONTROL Keywords] werden alle Keywords in der gefilterten Ansicht für das ausgewählte Advertiser-Konto aufgelistet.

* Die Registerkarte [!UICONTROL Negatives] listet alle negativen Keywords in der gefilterten Ansicht für das ausgewählte Advertiser-Konto auf.

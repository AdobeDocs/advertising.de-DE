# (Neue Benutzeroberfläche) Über die [!UICONTROL Placements]

*Beta-Funktion*

Die Ansicht [!UICONTROL Manage] > [!UICONTROL Placements] umfasst zwei Registerkarten:

* Die Registerkarte [!UICONTROL Placements] listet alle Platzierungen in der gefilterten Ansicht für das ausgewählte Advertiser-Konto auf.

* Die Registerkarte [!UICONTROL Negatives] listet alle negativen Platzierungen in der gefilterten Ansicht für das ausgewählte Advertiser-Konto auf.

# (Neue Benutzeroberfläche) Anhalten eines laufenden Bulksheet-Vorgangs

Sie können jeden Bulksheet-Vorgang stoppen - einschließlich einer Datei, die veröffentlicht wird - während er noch in Bearbeitung ist. Sie können auch Aufträge stoppen, die zwar geplant, aber noch nicht gestartet wurden.

1. Klicken Sie im Hauptmenü auf **[!UICONTROL Setup]** \> **[!UICONTROL Bulksheets]**.

1. Aktivieren Sie das Kontrollkästchen neben jeder Aufgabe, die Sie stoppen möchten.

1. Klicken Sie in der Aktionsleiste auf **[!UICONTROL Stop Job]**.

1. Klicken Sie in der Bestätigungsmeldung auf **[!UICONTROL Confirm]**.

>[!NOTE]
>
>Wenn Sie die Veröffentlichung einer Bulksheet-Datei abbrechen, nachdem einige Zeilen veröffentlicht wurden, wird eine EF-Fehler-Bulksheet mit den Zeilen generiert, die nicht veröffentlicht wurden.
